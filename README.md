## Technologies
*Ubuntu 18.04
*Node.js version 12.8.1
*npm version 6.10.2
*Angular version 11.2.7

## Setup
# `$ apt install nodejs`
# `$ apt install npm`
# `$ npm install -g @angular/cli`
# `$ git clone https://github.com/mixzafeir/Movie-Viewer.git`
# `$ cd [PROJECT_NAME]`.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
